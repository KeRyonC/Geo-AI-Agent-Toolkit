# Geo-AI-Agent-Toolkit
Input(DJI Images) -> MiMo Vision Encoder -> Semantic Masking -> Spatial Analysis -> Output(GeoJSON Alert Layer)
在测绘外业场景中，严重的几何处理已难以满足需求。当前主要难题为：针对无人机采集的地形复杂数据，如何从“测绘”转化为“业务决策图”。即：如何快速识别滑坡、违建或农田损毁等具体形象目标，并与GIS坐标系进行联动匹配。

该系统将原来的“数据处理时长”历时至秒级“业务洞察”。通过本次Token申请，我计划利用MiMo大模型的推理深度，进一步训练一个小型的空间逻辑增强模块，为后续进入大疆GIS研发岗位积累针对“地理信息+多态识别”的实战代码储备。
